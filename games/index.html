反恐精英 Counter-Strike
2019-11-18 / 童年回忆 神作 FPS / ★★★★★ 力荐

英雄联盟 League of Legends
2019-11-18 / MOBA 团队 策略 / ★★★★☆ 推荐
高中那会火的，班里男生基本都在玩，第一次和好友通宵就是在玩这个游戏，当时一边听着QQ音乐梁静茹陈绮贞五月天周杰伦的歌一边玩，现在回想起来很是有趣。
无主之地2 Borderlands 2
2019-11-18 / 神作 第一人称射击 角色扮演 / ★★★★★ 力荐
玩过好几遍的游戏，最开始和好友在网吧联机，后来上大学堕落那会自己在家一个人玩，枪械系统的设计很有特色。
孤岛危机2 Crysis 2
2019-11-18 / 童年回忆 神作 FPS / ★★★★★ 力荐
这个系列最开始玩的就是第二部了，剧情不错，难易度恰到好处，最喜欢这种可以有不同策略的游戏，我喜欢走潜行的路线。
孤岛危机 Crysis
2019-11-18 / 童年回忆 EA 第一人称射击 FPS / ★★★★☆ 推荐
初中毕业后那会，当时的好友推荐的，我们两在网吧里玩了几天通关的，可惜不能联机玩。
鬼泣3 Devil May Cry 3
2019-11-18 / 童年回忆 DMC 神作 动作 / ★★★★★ 力荐
初中的时候看哥哥玩的，神作。还记得当时PC版有的连招不太好放气得差点把键盘按坏。
侠盗猎车手：罪恶都市 Grand Theft Auto: Vice City
2019-11-18 / 童年回忆 沙盒 GTA / ★★★★★ 力荐
小时候基本都玩过的游戏，那时候谁去做什么任务，上来就是一辆坦克。
QQ飞车
2019-11-18 / 童年回忆 竞速 腾讯 / ★★★☆☆ 还行
小学六年级吧，暑假每天下午偷偷跑到家附近的网吧，每次去都几乎坐满了人，那是大家基本上都在玩qq飞车，我也经常和他们切磋，有时候也会有几个小朋友站在你后面看着你玩。
帝国时代2：国王时代 Age of Empires II: The Age of Kings
2019-11-18 / 童年回忆 即时战略 Microsoft / ★★★★★ 力荐
小学玩的，在当时的星际红警帝国中我选择了帝国时代。
地下城守护者 Dungeon Keeper
2019-11-18 / 童年回忆 神作 模拟经营 策略 牛蛙 / ★★★★★ 力荐
童年时玩的无可挑剔的神作，也是当时爸爸在玩然后安利给我的，通了几遍关仍意犹未尽。高中的时候下载了一些别人自己做的地图包玩，真的很有趣的一款游戏，虽然放到现在画质差了点，但丝毫不能掩盖它的优秀！
暗黑破坏神2 Diablo II
2019-11-18 / 童年回忆 暴雪 角色扮演 / ★★★★★ 力荐
小学五年级春节的时候，看着爸爸拿笔记本玩这个游戏，当时他好像开了修改器哈哈现在回想起来。 然后自己偷偷拷到了家里的台式机上。
零式赛车 最高时速 F-ZERO FOR GAMEBOY ADVANCE
2019-11-18 / 童年回忆 GBA / ★★★★☆ 推荐
小学时候玩的，当时一边听着周杰伦的听妈妈的话一边和基友玩赛车，现在想起来，真是无忧无虑。
宝可梦 绿宝石 ポケットモンスター エメラルド
2019-11-18 / GBA 口袋妖怪 童年回忆 神作 / ★★★★★ 力荐
不知道玩了有多少遍的游戏，从小学的GBA模拟器到高中手机上的模拟器；绝对零度这个词最开始还是从这个游戏里学到的。
黄金太阳：开启的封印 黄金の太陽 開かれし封印
2019-11-18 / GBA 童年回忆 神作 角色扮演 / ★★★★★ 力荐
小学四五年级的时候，家里买了电脑，哥哥给我拷了GBA模拟器和这个游戏，每天放学后回家和那时候的基友两个人一起玩；记得后面有一关实在不会解密卡了很久，到最后也没看到故事的结局。 那时候的基友现在也不知在天涯何处。
看门狗2 Watch_Dogs 2
2019-01-25 / 沙盒 Ubisoft 动作 / ★★★★★ 力荐
很久没碰到这样一个能让我废寝忘食玩一天的游戏了；虽说玩的游戏越多就越挑剔，但看门狗2给我的感受是当一个游戏的氛围做的恰到好处之后，一些细枝末节的缺点便不是那么重要。 穿梭在旧金山的街头，光是听着古典乐漫步，就很有感觉
